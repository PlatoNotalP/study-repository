# Команды git
1. **Конфиг git:**
- git config --global user.email "my.mail@gmail.com"
- git config --global user.name "Notalp"
- git config --list
2. **Инициализация локального репозитория:**
- git init
3. **Общая сводка по репозиторию:**
- git status
4. **Добавление файлов в предстоящий commit:**
- git add <название файла>
5. **Создание коммита:**
- git commit -m "<сообщение>"
6. **Просмотр коммитов:**
- git log
7. **Просмотр изменений:**
- git diff
8. **Откат изменений до последнего коммита:**
- git restore <название файла>
9. **Отправка изменений на удаленный репозиторий:**
- git push
10. **Создание новой ветки:**
- git branch <название ветки>
11. **Просмотр веток:**
- git branch
12. **Переключение на ветку:**
- git checkout <название ветки>
13. **Добавление последних изменений в локальный репозиторий**
- git pull
14. **Копирование удаленного репозитория на локальную машину**
- git clone <ссылка на репозиторий>
